.. note::

  Ensure a valid Solace Cloud Token is defined for Event Portal V1 and Event Portal V2.
  They can both be the same.


  .. code-block:: bash

    export EP_V1_SOLACE_CLOUD_TOKEN={your token}
    export EP_V2_SOLACE_CLOUD_TOKEN={your token}
